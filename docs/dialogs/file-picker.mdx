## Quick start

The File Picker component provides a the native interface for users to browse and select files from their device. It supports various file types and offers customization options to fit different use cases.

<CodeGroup>

```kotlin filekit-dialogs
val file = FileKit.pickFile()
```

```kotlin filekit-dialogs-compose
val launcher = rememberFilePickerLauncher { file ->
  // Handle the file
}

Button(onClick = { launcher.launch() }) {
  Text("Pick a file")
}
```

</CodeGroup>
